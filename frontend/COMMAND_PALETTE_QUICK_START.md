# Command Palette - Quick Start Guide

## What is the Command Palette?

The Command Palette is a powerful keyboard-driven search and navigation system that allows you to quickly access any page in the application without using your mouse.

## How to Use

### Opening the Command Palette

**Mac:** Press `⌘ + K` (Command + K)
**Windows:** Press `Ctrl + K`

Or click the search button in the top-left corner of the application.

### Searching

Simply start typing to search. The command palette supports:

- **Exact matches**: Type "Dashboard" → finds Dashboard
- **Partial matches**: Type "coa" → finds "Chart of Accounts"
- **Keywords**: Type "clients" → finds "Customers"
- **Modules**: Type "Accounting" → shows all Accounting pages

### Navigation

- **↑ / ↓** - Move through search results
- **Enter** - Navigate to selected page
- **Esc** - Close the command palette
- **Home / End** - Jump to first/last result

### Favorites

1. Open the command palette
2. Hover over any page
3. Click the ☆ icon to add to favorites
4. Click the ★ icon to remove from favorites

Favorites appear at the top for quick access.

### Recent Items

The command palette automatically tracks the 5 pages you've visited most recently. These appear right after your favorites.

## FAQ

**Q: How many pages can I favorite?**
A: Unlimited! There's no limit to the number of favorites.

**Q: Where are my favorites saved?**
A: They're saved in your browser's localStorage, so they persist across sessions.

**Q: Will this work on mobile?**
A: Yes! The command palette works on mobile devices, though you'll need to tap the search button instead of using keyboard shortcuts.

**Q: What if I forget the keyboard shortcut?**
A: Just click the search button in the top-left corner of the application. The shortcut is displayed there too.

**Q: Can I customize the keyboard shortcut?**
A: Currently, `⌘K`/`Ctrl+K` is the global standard for command palettes (used by GitHub, Slack, Linear, etc.). Customization may be added in the future.

**Q: Why do some pages say "Coming Soon"?**
A: These pages are planned but not yet implemented. You'll see a friendly toast notification if you try to navigate to them.
